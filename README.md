# CrackPost_connector

## 项目简介

CrackPost_connector 是为 CrackPost 企划组定制开发的个人用户通信历史可视化工具，由个人独立维护。本程序不涉及任何个人隐私的非法使用，有兴趣可参考
源码，所有操作可独立完成。

## 主要功能

- 邮件批量下载与去重，自动归档通信内容及附件
- 智能识别信件类型（A/B/C），自动补全收发人信息
- 支持多邮箱平台（QQ、126、163、Gmail、Outlook 等）
- 可视化人际关系网络（基于 HTML5 Canvas，支持时序播放、节点拖拽、全文弹窗等）
- 配置化检索规则，支持 YAML/环境变量优先级覆盖
- 详细日志与下载报告，便于后续分析和溯源
- 插件扩展点，支持自定义认证、格式化、传输策略

## 快速开始

1. 通过dist内程序打开gui，并按照提示说明下载文档。
2. 若已下载好数据，可通过单击start.bat打开网页。

## 配置说明

- 支持环境变量或 config.yml 配置
- 邮箱授权码需在邮箱设置中获取（搜索如何开启 IMAP 并生成授权码，授权范围调至全部）
- 如需帮助请联系`1152254208@qq.com`

## 可视化功能

- 打开 visualization_private.html 即可浏览通信网络关系图
- 支持时序播放、节点拖拽、信件全文弹窗、字数统计等高级交互

## 文件说明

- `backup`  存储备份和过往版本的下载与html脚本，待未来全局可视化使用
- `dist`    编译好的程序，可以直接打开使用
- `font`    本程序用到的字体
- `icon`    企划图标存储
- `venv`    用于打包的虚拟环境
  
- `download_gui.py`             下载界面源代码
- `visualization_private.html`  个人可视化网页源代码
